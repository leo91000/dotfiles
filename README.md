# Dotfiles

My personnal dotfiles folder to use.

## Requirements

- Neovim
- Python 3.10+ & pip
- Repositories :
  - Tmux plugin manager : `git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm`
  - P10k : `git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ~/powerlevel10k`

## First time using neovim

- `:PackerInstall`
- `:CHADdeps`

## Updating

- `:PackerUpdate`
- `:TSUpdate`
- `:CHADdeps`
